Science JSON

Introduction

Science JSON is a generic data structure format for describing different types of data samples. It was created for Science Hack Day Guelph 2013 by Edward Kim and John Yeung.

Explanation

Science JSON is a fairly open format, and has fields with data types that can be used to describe a wide array of data structures. See the table below.


FieldRequired?Data Type
Experimenter NameYesString
Sample NameYesString
DateYesDictionary (DateType : Date)
ProcedureNoString (long)
Experimental ConditionsNoDictionary (Condition : Value)
ApparatusYesString (array)
Chemical ConstituentsNoString (array)
Other ConstituentsNoString (array)
FiguresNoString/URL (array)

Example

{
	'exp_name' : 'John Doe',
	'sample_name' : 'sample_001',
	'date' : {
		'fabrication_date' : '2010-10-04',
		'experiment_date' : '2011-10-08'
	},
	'apparatus' : [
		'optical microscope',
		'scanning electron microscope'
	],
	'exp_cond' : {
		'temperature' : '298K'
	}
}
